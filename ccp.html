Smart Overheating Protection System

This project monitors the temperature of household and industrial appliances like washing machines, refrigerators, and motors to prevent overheating. When the temperature exceeds a safe threshold, the system automatically disconnects the power supply to ensure safety and appliance longevity.

How It Works

The Smart Overheating Protection System continuously monitors temperature using a DS18B20 sensor. The ESP32 microcontroller processes the temperature data and controls a relay that cuts power to the appliance if the temperature exceeds the set threshold. Users can also monitor and control the system remotely via a web interface.
